# Radar X – Diagnóstico Inteligente de Churn (D-90)  Plataforma avançada de análise de churn que combina **IA Generativa (Multiagentes)**, **LangGraph** e **Machine Learning** para identificar a causa raiz do cancelamento a partir de jornadas D-90 (90 dias antes do churn).
